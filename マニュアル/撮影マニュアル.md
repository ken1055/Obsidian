# 撮影マニュアル

[[事業戦略]] 5 章の現場手順を運用ドキュメントとして切り出したもの。撮影前に必ずこのチェックリストを通す。

## 1 物件の標準工数

| 工程 | 時間 |
|---|---|
| 移動・準備 | 0.5 h |
| 撮影 | 1 h |
| データ処理 (3DGS 変換) | 2 h |
| 確認・納品 | 0.5 h |
| **合計** | **4 h** |

## 撮影前チェックリスト

- [ ] カーテン・ブラインドを調整 (自然光を最大化)
- [ ] 蛍光灯・照明をすべて点灯
- [ ] 不要な荷物・段ボールを一時的に隠す
- [ ] Portalcam アプリでプレビュー確認してから本撮影開始
- [ ] 窓の逆光が強い場合はポータブル LED パネルで補光

## 撮影後・現地離脱前チェック (重要)

- [ ] アプリで大まかな品質チェック
- [ ] NG 箇所はその場で再撮 (帰宅後に発覚するのが最悪)

## NG が出た場合のリカバリー

| 状況 | 対応 |
|---|---|
| 部分的な品質低下 | ビューアの初期視点を品質が良いエリアに設定して目立たなくする |
| 極端な失敗箇所 | その部屋のみ 360 度写真で補完 (ハイブリッド) |
| 全体的に撮り直し | 契約に明記した「1 回限り無料再撮」を発動 |

## 納品時のセット

- [ ] ビューア URL を発行
- [ ] UTM / 固有 URL で計測コード埋め込み済みか確認
- [ ] 仲介業者へ掲載手順を共有 (SUUMO コメント欄 / QR 合成 / 自社サイト / at home / Homes の経路ごと)
- [ ] 「成約後のデータ削除タイミング」を口頭でも再確認
